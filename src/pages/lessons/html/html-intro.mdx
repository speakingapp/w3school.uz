# HTML - HyperText MarkUp Language

> **HTML** (HyperText Markup Language) -  gipermatnli belgilash tilini bildiradi.
HTML bizga hujjat yoki veb-sayt tuzilishini aniqlash imkonini beradi.
HTML bu **dasturlash tili EMAS**, bu belgilash tili, ya'ni uning maqsadi algoritmni aniqlash emas, balki veb-sayt tarkibiga tuzilish berishdir.
Bu barcha veb-sayt ma'lumotlarini (masalan, matnlar, tasvirlar va videolar) o'z ichiga oladi.
---

### HTMLning umumiy tuzilish

```html
<!DOCTYPE html>
<html>
<head>
	<title>Hujjat nomi</title>
</head>
<body>

</body>
</html>
```

#### Hujjat tahlili:

- `<!DOCTYPE html>` deklaratsiyasi ushbu hujjat HTML5 hujjati ekanligini belgilaydi
- `<html>` elementi HTML sahifasining ildiz elementidir
- `<head>` elementi HTML sahifasi haqidagi meta-ma'lumotni o'z ichiga oladi
- `<title>` elementi HTML sahifasi uchun sarlavhani belgilaydi (bu brauzerning sarlavha satrida yoki sahifa yorlig'ida ko'rsatilgan)
- `<body>` elementi hujjatning tanasini belgilaydi va sarlavhalar, paragraflar, rasmlar, giperhavolalar, jadvallar, ro'yxatlar va boshqalar kabi barcha ko'rinadigan tarkiblar uchun konteynerdir.
- `<p>` elementi paragrafni belgilaydi
